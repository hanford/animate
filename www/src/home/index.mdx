## How to

Install any animation as a dependency using yarn
```bash
yarn add @animate/bounce
```

or npm
```bash
npm install @animate/bounce --save
```

Finally, wrap your component! 
```javascript
import * as React from 'react'
import Bounce from '@animate/bounce'

..

 render () {
   return (
     <Bounce>
      <Text>I'm animated</Text>
    </Bounce>
   )
 }
```
