# Bounce

First start off by installing the `bounce` package via yarn

```bash
yarn add @animate/bounce
```

or npm

```bash
npm i @animate/bounce --save
```

Then simple import the component and you're all set!

import { Playground } from 'docz'
import {Bounce} from '../../../../packages/bounce'

## `<Bounce />`

<Playground>
  <Bounce>
    <div
      style={{
        maxWidth: 300,
