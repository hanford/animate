# blur

First start off by installing the `blur` package via yarn

```bash
yarn add @animate/blur
```

or npm

```bash
npm i @animate/blur --save
```

Then simple import the component and you're all set!

import { Playground } from 'docz'
import Blur from '../../../../packages/blur'
